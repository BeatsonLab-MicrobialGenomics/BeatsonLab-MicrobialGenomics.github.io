Software

Our group has developed a number of software tools that simplify the analysis of next generation sequencing data. Much focus has been on visualisation.

Published

BRIG

BLAST Ring Image Generator. BRIG is a cross-platform (Windows/Mac/Unix) application that can display circular comparisons between a large number of genomes, with a focus on handling genome assembly data.

Please cite the BRIG paper if BRIG is used to generate figures for publications: NF Alikhan, NK Petty, NL Ben Zakour, SA Beatson (2011) BLAST Ring Image Generator (BRIG): simple prokaryote genome comparisons, BMC Genomics, 12:402. PMID: 21824423

EasyFig

Easyfig is a Python application for creating linear comparison figures of multiple genomic loci with an easy-to-use graphical user interface (GUI).

If Easyfig is used to generate figures for publication, please cite our paper: Sullivan MJ, Petty NK, Beatson SA. (2011) Easyfig: a genome comparison visualiser. Bioinformatics; 27 (7): 1009-1010. PMID: 21278367

Pre-publication

SeqFindR

SeqFindR is a tool to easily create informative genomic feature plots.

If you used to generate figures for publication please cite like this: Stanton-Cook M, Alikhan NF, Forde BM, Ben Zakour NL and Beatson SA (2014) SeqFindR - easily create informative genomic feature plots http://github.com/mscook/SeqFindR
